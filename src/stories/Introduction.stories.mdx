# ![](/seeelaye.svg)

see·el·aye, _as in CLI_, is a terminal-emulator-emulator for the browser,
written in [Vue](https://v3.vuejs.org).

While it aims to provide the closest approximation of a real terminal in
the browser, see·el·aye runs completely on the front-end of the application and
does not require a host or an actual file system.
