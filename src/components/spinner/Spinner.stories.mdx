## Styling

The component can be styled by setting the following CSS custom properties.

| Property               | Description                      | Default                |
|------------------------|----------------------------------|------------------------|
| `--spinner-edge-color` | Color of the end brackets        | `--color-secondary-fg` |
| `--spinner-char-color` | Color of the moving progress bar | `--color-normal-fg`    |
