## Styling

The component can be styled by setting the following CSS custom properties.

| Property         | Description                 | Default                  |
|------------------|-----------------------------|--------------------------|
| `--folder-color` | Text color for folder links | `--color-normal-cyan`    |
| `--file-color`   | Text color for file links   | `--color-normal-magenta` |
| `--alias-color`  | Text color for node aliases | `--color-secondary-fg`   |
