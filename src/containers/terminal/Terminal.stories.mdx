## Theming

The terminal component supports theming using the `class` attribute. It ships
with my favourite themes out-of-the-box, but you can surely
[define custom themes](?path=/docs/guide-custom-themes--page) and add as many as
you like.

### Solarized dark

**Classes:** `solarized`, `dark`
